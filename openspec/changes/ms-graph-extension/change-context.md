# Change Context: ms-graph-extension

This change (P5) replaces four Microsoft 365 extension stubs (`ms_graph`,
`outlook`, `teams`, `sharepoint`) with real implementations and lifts the
MS Agent Framework harness from `NotImplementedError` to a fully-wired
`SdkHarnessAdapter` over the `agent-framework` SDK. It introduces a
reusable foundation — `CloudGraphClient` Protocol, two `MSALStrategy`
implementations, and an httpx-based `GraphClient` with resilience,
observability, and Retry-After / 401-refresh / cross-domain-redirect
handling — that P14 google-extensions can satisfy with its own concrete
clients.
