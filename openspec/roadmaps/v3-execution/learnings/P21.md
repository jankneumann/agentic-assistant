# Learning: P21

## Decisions
- **Sync-to-async bridge for snippet retrieval**: _run_blocking: asyncio.run when no loop; single-worker-thread asyncio.run when inside a running loop; cross-loop pool errors treated as degraded memory ([] + WARNING), never fatal
  - Rejected: making get_recent_snippets async (breaks MSAF D27 call site)
  - Rejected: nest_asyncio
- **Capture awaited-inline, not create_task-detached**: CLI drives turns with asyncio.run so detached tasks die with the per-turn loop; never-breaks-conversation comes from swallow-and-warn in the shared SdkHarnessAdapter._capture_interaction helper
- **Streaming capture placement**: Captured from concatenated TextDelta before the terminal success RunFinished — code after the last yield is skipped on GeneratorExit

## Recommendations
- Watch asyncpg pool churn on the worker-thread bridge path; loop-affine engine strategy if real usage shows errors
- Graphiti episode write-back on capture is deferred — P28 candidate
- Agent worktrees branch from session-start HEAD, not current branch head — plan a rebase/cherry-pick integration step for every worktree agent
