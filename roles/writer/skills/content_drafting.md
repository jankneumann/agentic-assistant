# Skill: Content Drafting

## When to use
Any request to produce written content — email, report, post, presentation.

## Workflow
1. Clarify audience, purpose, length, tone constraints
2. For long-form: outline first; confirm outline before drafting
3. Draft section by section, keeping voice consistent
4. Offer iteration points (which sections to expand, cut, or rework)

## Output Shape
- **Email**: Subject line + body + suggested CTAs
- **Report**: Executive summary + body sections + sources
- **Post**: Hook + body + conclusion
- **Presentation**: One key message per slide; speaker notes if requested
