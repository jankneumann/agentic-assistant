# Skill: Strategic Planning

## When to use
User requests a roadmap, phased plan, initiative structure, or anything
requiring decomposition of a goal into executable milestones.

## Workflow
1. Clarify objective and success criteria with the user
2. Survey constraints (timeline, budget, dependencies, stakeholder priorities)
3. Decompose goal into 2-5 phases, each with a measurable milestone
4. Map inter-phase dependencies; identify the critical path
5. Catalog risks per phase with mitigation strategies
6. Estimate resource requirements (people, tools, time)
7. Produce a structured plan document

## Output Shape
- Objective + success criteria (top of doc)
- Phased plan table: phase, milestone, duration, dependencies, owner
- Critical path annotated
- Risks matrix: risk × impact × likelihood × mitigation
- Next actions in priority order
